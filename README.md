# Mfcollins3 Tap

## How do I install these formulae?

`brew install mfcollins3/tap/<formula>`

Or `brew tap mfcollins3/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mfcollins3/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
